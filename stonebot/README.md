# 디스코드 돌팔이 봇의 설명서 입니다.  
## 모든 기능은 현금과 전혀 관계가 없으며 재미로만 이용해주세요.  
  
서버 내 이름 변경과 타임아웃 기능이 있으며 정상적인 작동을 위해 관리자 권한 및 최상위 역할 설정, 이를 원치 않을 경우 최하위 권한 설정이 필요합니다.  
돌멩이를 베팅하는 대부분의 게임은 모두 유저에게 유리하도록 세팅되어있습니다.  
진지하게 하지 마시고 가볍게 즐겨주세요.  

## 돌팔이 봇 이용 약관
1. 돌팔이 봇은 재미를 최우선으로 생각하며 사용자들에게 재미를 줄 목적으로 만들어졌습니다.  
2. 돌팔이 봇의 모든 기능은 현금과 전혀 관련이 없으며 단순 재미를 위한 기능입니다.  
3. 돌팔이 봇의 사용으로 인해 발생한 피해나 손해에 대해 개발자는 책임지지 않습니다. 다만 피해나 악용이 우려되는 부분에 대해 언제든 제보 가능합니다.  
4. 돌팔이 봇의 서비스를 위해 최선을 다하지만, 예기치 못한 문제로 인해 발생하는 봇의 데이터 손실에 대해 개발자는 책임지지 않습니다.  
5. 돌팔이 봇을 악용하거나 기타 커뮤니티에 피해가 가는 행동이 제보되거나 감지될 경우, 유저는 봇 사용이 제한될수 있습니다.  
  
**개인정보 수집**  
1. 돌팔이 봇은 원활한 운영을 위해 다음과 같은 개인정보를 저장합니다.  
- 서버 고유 ID, 채널 고유 ID (공지 서비스를 위한 데이터)  
- 유저의 고유 ID (유저의 돌멩이 관련 데이터 저장)  
- 유저의 돌팔이 봇 명령어 사용 기록 (명령어 사용 횟수, 사용 시간 등)  
2. 돌팔이 AI는 다음과 같은 개인정보를 수집하며 55일 뒤에 자동으로 파기됩니다.  
- 유저의 서버 내 닉네임  
- 유저의 AI 채팅 내역 (질문과 답변)  
3. 그 외 모든 데이터는 별도로 수집하거나 저장하지 않습니다.  
4. 모든 개인정보 관련 데이터는 서비스 제공을 위해 이용 기간 동안만 보관되며, 서비스 종료시 파기됩니다.  
5. 수집된 모든 데이터는 AI 학습에 사용되지 않습니다.    
6. 모든 유저는 본인이 원할때 개발자에게 연락해 개인정보 관련 데이터를 삭제할 수 있습니다.  
  
**기타 문의 및 제보**  
돌팔이 봇의 불쾌한 부분이나 개선이 필요한 사항, 개인정보 삭제 관련 사항은 언제든지 /제보 명령어를 이용해 개발자에게 제보가 가능합니다.

## /제보  
개발자에게 봇의 버그나 개선사항, 추가되면 좋겠다 싶은 아이디어를 전송합니다.  
스팸 및 도배시 해당 기능이 차단될수도 있습니다.  

## /사용설정 [옵션]  
어느 채널에서 봇을 사용하고 금지할지 설정합니다.  
서버 관리자와 기타 권한이 있는 유저는 이 제한을 무시합니다.  
### [설정 옵션]  
 - 없음 : 현재 서버의 모든 채널에서 봇 사용을 허용합니다.  
 - 현재 채널 금지 : 현재 채널에서 봇 사용을 금지합니다.  
 - 현재 채널 허용 : 현재 채널에서 봇 사용을 허용합니다.  
 - 현재 채널만 허용 : 현재 채널을 제외한 서버의 모든 채널에서 봇 사용을 금지합니다.  
 - 현재 서버 AI 비활성화 : 현재 서버에서 AI 답변을 금지합니다.  
 - 현재 서버 AI 활성화 : 현재 서버에서 AI 답변을 허용합니다.  
  
## /공지설정 [게임]  
이 명령어를 입력한 채널로 해당 게임의 점검 공지를 설정합니다.  
점검 공지가 감지되거나 점검이 완료될 경우 해당 채널에 공지를 올립니다.  
### [현재 알림 서비스 중인 게임]  
 - 마비노기 모바일  
  
## /공지해제  
해당 채널에 등록된 모든 공지를 해제합니다.  
  
## / 한강  
서울시의 수질 측정 시스템을 이용해 현재 한강의 수온과 성분을 자세히 알려줍니다.  
  
## /운세  
돌팔이가 측정한 오늘의 운세를 보여줍니다.  
확률과 연관은 없습니다.  
   
## /돌멩이  
현재 돌멩이와 기록을 보여줍니다.  
  
## /배급  
10~100의 랜덤한 돌멩이를 배급받습니다.  
12시간에 한번 사용 가능합니다.  
  
## /구제  
서버에 있는 친구들에게 구제 요청을 합니다.  
친구들이 도와주기를 눌렀을 경우 돌멩이 20개 획득,  
어쩔티비를 눌렀을 경우 돌멩이를 20개 잃습니다.  
친구들은 어떤 버튼을 눌러도 아무런 이득과 손해를 보지 않습니다.  
현재 보유 돌멩이가 0개일 경우에만 사용 가능하며 6시간에 한번 사용 가능합니다.  
  
## /베팅 [돌멩이]  
[돌멩이] 만큼 베팅합니다.  
실패시 돌멩이를 잃으며 성공시 돌멩이를 얻습니다.  
성공확률은 `51%`이며 성공시 `10%` 확률로 2배로 돌멩이를 얻습니다. 
  
## /슬롯 [돌멩이]  
[돌멩이] 만큼 투자하여 슬롯머신을 돌립니다.  
10 이상 1000이하의 돌멩이를 넣어야 합니다.  
확률은 아래에 기재되어 있습니다.  
  
## /선물 @대상 [돌멩이]  
@대상 에게 [돌멩이] 만큼 선물합니다.  

## /뿌리기 [옵션] [돌멩이]  
50 이상의 돌멩이를 뿌립니다.  
선착순 - 가장 먼저 받기 버튼을 누른 사람에게 돌멩이를 뿌립니다.  
랜덤 - 서버 내 랜덤한 사람에게 돌멩이를 뿌립니다. 본인이 받을 확률도 있습니다.  
거지 - 서버 내 포인트가 가장 적은 사람에게 돌멩이를 뿌립니다.  
  
## /강탈  
서버내 돌멩이를 100개 이상 소유한 랜덤한 대상을 상대로 강탈을 시도합니다.  
강탈 성공시 대상이 소유한 돌멩이의 `10~50%`를 훔쳐옵니다.  
역강탈 당할 경우 반대로 본인이 소유한 돌멩이의 `10~50%`를 상대에게 빼앗깁니다.  
최대 강탈 가능 돌멩이는 본인이 소유한 돌멩이의 두배를 넘지 못합니다.  
한시간에 한번 사용 가능합니다.  
*강탈 성공/실패/역강탈 확률 = 각각 `33.33%`  
*강탈 성공시 스스로를 강탈할 확률이 존재합니다  
  
## /룰렛  
5개의 약실과 1발의 총알로 러시안 룰렛을 시작합니다.  
라운드가 진행될수록 얻는 돌멩이가 늘어나며 실패시 모든 보유 돌멩이를 잃습니다.  
1라운드 : `30%`  
2라운드 : `40%`  
3라운드 : `50%`  
4라운드 : `100%`  
  
## /블랙잭 [돌멩이]  
[돌멩이] 만큼 베팅해서 블랙잭을 합니다.  
스플릿은 없으며 이기면 베팅한 만큼 얻고 지면 잃습니다.  
블랙잭 성공시 1.5배를 얻습니다.  
  
## /바카라 [돌멩이]  
[돌멩이] 만큼 베팅해서 바카라를 합니다.  
플레이어와 딜러, 타이에 베팅 가능하며 이기면 베팅한 만큼 얻고 지면 잃습니다.  
타이에 베팅하여 이겼을 경우 8배를 얻습니다.  
  
## /홀덤 [돌멩이]  
[돌멩이] 만큼 베팅해서 홀덤을 합니다.  
포커의 족보를 사용하고 기본적인 텍사스 홀덤 룰을 살짝 변형해서 적용합니다.  
레이즈시 현재까지 베팅한 돌멩이의 `50%`를 추가로 베팅해서 판돈을 올립니다.  
폴드시 현재까지 베팅한 돌멩이의 `10%`를 회수하고 나머지를 잃습니다.  

## /복권 [번호]  
10000 ~ 99999 의 5자리 랜덤한 번호로 즉석 복권을 시도합니다.  
20돌멩이를 소모하며 당첨 실패시 당첨액에 40 돌멩이를 누적시킵니다.  
당첨액은 1,000,000 부터 시작하며 누적된 당첨 돌멩이도 포함됩니다.  
  
## /랭킹  
서버내 유저들의 돌멩이 랭킹을 봅니다.  
0돌멩이를 가진 유저는 보이지 않습니다.  
  
## /이름 @대상 [이름]  
2000돌멩이를 소모해 @대상의 서버내 이름을 마음대로 바꿉니다.  
사전에 서버내 봇의 역할이 최상단에 위치되어야 하며 서버 소유자는 대상으로 설정 불가능합니다.  
  
## /마스크 @대상  
3000돌멩이를 소모해 @대상에게 1분 타임아웃을 걸어버립니다.  
사전에 서버내 봇의 역할이 최상단에 위치되어야 하며 서버 소유자는 대상으로 설정 불가능합니다.  
  
## /벨튀 @대상  
3000 돌멩이를 소모해 서버내 @대상에게 익명으로 짧은 DM을 보냅니다.  
!와 ?를 제외한 특수문자가 첨부된 메세지는 전송 불가능합니다.  
  
## /상태 [메세지]  
5000돌멩이를 소모해 봇의 온라인 상태 메세지를 변경합니다.  
!와 ?를 제외한 특수문자가 첨부된 메세지는 전송 불가능합니다.  
부적절한 내용이 담길 경우 초기화 됩니다.

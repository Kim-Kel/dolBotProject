# 디스코드 돌팔이 봇의 설명서 입니다.  


## /포인트  
현재 포인트와 기옥을 보여줍니다.  

## /배급  
10~50의 랜덤한 포인트를 배급받습니다.  
12시간에 한번 사용 가능합니다.  

## /베팅 [포인트]  
[포인트] 만큼 베팅합니다.  
실패시 포인트를 읽으며 성공시 포인트를 얻습니다.  
성공/실패 확률은 각각 `50%`이며 성공시 `10%` 확률로 2배로 포인트를 얻습니다.  

## /선물 @대상 [포인트]  
@대상 에게 [포인트] 만큼 선물합니다.  

## /강탈  
서버내 100포인트 이상의 랜덤한 대상을 상대로 강탈을 시도합니다.  
강탈 성공시 대상이 소유한 포인트의 `10~50%`를 훔쳐옵니다.  
역강탈 당할 경우 반대로 본인이 소유한 포인트의 `10~50%`를 상대에게 빼앗깁니다.  
최대 강탈 가능 포인트는 본인이 소유한 포인트의 두배를 넘지 못합니다.  
강탈 성공/실패/역강탈 확률 = 각각 `33.33%`  

## /랭킹  
서버내 유저들의 포인트 랭킹을 봅니다.  
0포인트를 가진 유저는 보이지 않습니다.  

## /이름 @대상 [이름]  
1000포인트를 소모해 @대상의 서버내 이름을 마음대로 바꿉니다.  
사전에 서버내 봇의 역할이 최상단에 위치되어야 하며 서버 소유자는 대상으로 설정 불가능합니다.  

## /마스크 @대상  
1000포인트를 소모해 @대상에게 1분 타임아웃을 걸어버립니다.  
사전에 서버내 봇의 역할이 최상단에 위치되어야 하며 서버 소유자는 대상으로 설정 불가능합니다.  

## /나가 @대상  
500포인트를 소모해 @대상을 통화방에서 쫓아냅니다.  
사전에 서버내 봇의 역할이 최상단에 위치되어야 하며 서버 소유자는 대상으로 설정 불가능합니다.  

## /벨튀 @대상  
2000 포인트를 소모해 서버내 @대상에게 익명으로 짧은 DM을 보냅니다.  
URL이 첨부된 메세지는 전송 불가능합니다.  

## /상태 [메세지]  
3000포인트를 소모해 봇의 온라인 상태 메세지를 변경합니다.  
부적절한 내용이 담길 경우 초기화 됩니다.
